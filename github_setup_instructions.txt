GITHUB SETUP INSTRUCTIONS FOR LADLEANDTHEWHISK

1. Go to https://github.com/new
2. Repository name: ladleandthewhisk
3. Description: A modern, responsive website for Ladle and the Whisk, a culinary store in New Orleans
4. Make it Public
5. Don't initialize with README (since we already have one)
6. Click "Create repository"

7. Run these commands in your terminal:
   git remote add origin https://github.com/jvalamis/lwhisk.git
   git push -u origin main

8. To set up GitHub Pages:
   - Go to your repository on GitHub
   - Click on "Settings"
   - Scroll down to "GitHub Pages" section
   - Under "Source", select "main" branch
   - Click "Save"
   - Wait a few minutes for your site to be published
   - Your site will be available at https://jvalamis.github.io/lwhisk/

If you prefer to use SSH instead of HTTPS for GitHub, use this command instead:
   git remote add origin git@github.com:jvalamis/lwhisk.git
